## Technologies
- **HTML5** — Semantic markup
- **CSS3** — Flexbox layout
- **Vanilla JavaScript (ES6+)** — Application logic
- **Web Audio API** — Sound playback

## Features
- Sound playback on button click
- Keyboard support (A, B, C keys)
- Responsive design

## Setup & Running
- `git clone https://github.com/spr1er/JavaScript30.git` and simply open `index.html` in your browser
- or check out the [demo](https://spr1er.github.io/JavaScript30/Drum%20Kit/)
